# Anti-Patterns — The Generic "AI Voice" to Avoid

These are the universal tells that make a draft read as machine-written. None of them are personal to you — they're the default sludge every model drifts toward. The drafter should treat this list as hard banned unless a sample in `examples.md` proves you actually write this way.

## Banned phrases
- "I hope this email finds you well" (and every variant)
- "I wanted to reach out" / "I'm reaching out"
- "In today's fast-paced world" / "In an ever-evolving landscape"
- "Let's circle back" / "touch base" / "synergy" / "leverage" / "robust" / "seamless"
- "delve into" / "navigate the complexities of" / "a testament to"
- "I'd love to..." as filler enthusiasm
- "Don't hesitate to reach out"

## Banned structural moves
- **The hollow tricolon:** "clear, concise, and compelling." Three-part lists for rhythm, not meaning.
- **"It's not just X — it's Y."** The model's favorite false-profundity template.
- **Throat-clearing openings.** Two sentences of warm-up before the actual point.
- **Manufactured enthusiasm.** "So excited!", "Amazing!", exclamation marks you wouldn't use out loud.
- **Em-dash overuse.** One model crutch — used like this — in every single sentence.
- **Even, balanced paragraphs.** Real people write a one-liner, then a long one, then a fragment. Models write three tidy equal blocks.
- **The summary closer.** "In conclusion, ..." restating what was just said.

## The test
If you'd be slightly embarrassed to have written a line out loud to the recipient's face, it's probably an anti-pattern. Cut it.
