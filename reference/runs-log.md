# Runs Log

One line per real draft Voice Engine produced that I actually used. This is the ledger that makes the core promise — *"drafts need less editing over time"* — **measurable instead of asserted.** The corrections-log records *what* I fixed; this records *how much* I had to fix, run over run. When the edit distance trends toward "None," the folder is working.

Borrowed shape: Charlie Weeks' Marshal and Roc Lee's Daily Sonar — a running record so a claim about a trend has receipts.

**Edit distance scale:**
- **None** — sent as-is, zero changes. (These are also your best candidates to promote into `examples.md`.)
- **Light** — a word or link swap. Under ~10 seconds of editing.
- **Moderate** — reworked a sentence or added a missing fact.
- **Heavy** — rewrote most of it. If this repeats for one email type, that type needs a new rule or gold example.
